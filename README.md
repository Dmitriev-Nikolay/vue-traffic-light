# vue-traffic-light

## Project download
```
clone a repo:
https://github.com/Dmitriev-Nikolay/vue-traffic-light.git

and cd to /vue-traffic-light/
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
